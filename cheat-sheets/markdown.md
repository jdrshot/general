# MARKDOWN

<br>

## BASIC SYNTAX

|   Element   |   Markdown Syntax   |
| ---- | ---- |
| Headers | #, ##, ### |
| **Bold** | ** bold text ** |
| *Italic* | * italic text * |
| Blockquote | > blockquote |
| Ordered List | 1. Item 1 <br> 2. Item 2 <br> 3. Item 3 |
| Unordered List | - Item <br> - Item <br> - Item |
| Code | \` code \`|
| Horizontal Rule | --- |
| Link | \[title](url) |
| Image | ![alt text](image source) |
